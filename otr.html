guitarProjection

desenvolvido para Toca (2024)

Código para o performer de luzes/projeção de Toca.

Permite o mapeamento de caminhos lineares padronizados por onde os spotlights da peça vão passar previamente à performance. Utilizando uma controladora e a imagem de câmera infravermelha como feedback, o performer pode escolher a ordem dos caminhos que vai utilizar de acordo com cada momento da peça, podendo modficar em tempo real a posição horizontal cada luz em relação à cada caminho, um offset vertical, rotação e transposição dos caminhos para ajustar à performance do violonista. É também possível alterar a cor das luzes, o tamanho de seu foco, sua opacidade, etc.

Desenvolvido em Python e Processing 4. A documentação deste código pode ser encontrada aqui.

caminhooculto

desenvolvido para a residência SomaRumor (2021)

Sistema interativo baseado no jogo Caminho Oculto. Ao invés de peças de diferentes formatos que devem ser movidas para retirar uma peça específica, caminhooculto utiliza 8 LDRs e peças de plástico com diferentes transparências para identificar padrões randomizados em sua superfície. Cada padrão controla processamentos de audio distintos, também randomizados de tempo em tempo.

Desenvolvido em Arduino e SuperCollider. Documentação pode ser encontrada aqui.

joyOSCMapper

visualizador e mapeador de inputs de joycons para OSC (2021)

Visualizador e mapeador de inputs de Nintendo Switch Joycon's para OSC, permitindo a integração com sistemas interativos diversos.

Desenvolvido em C++/openFrameworks para a peça digdigjoy, que não saiu (ainda) do papel. Aqui você encontra a documentação.
